# Bataille Navale

## Objectif du Jeu
L'objectif du jeu Bataille Navale est de couler tous les navires de votre adversaire avant qu'il ne coule les vôtres.

## Matériel
- Deux grilles par joueur : une pour positionner ses propres navires et une pour noter les tirs effectués sur les navires adverses.
- Cinq types de navires par joueur :
  - Porte-avions (5 cases)
  - Cuirassé (4 cases)
  - Croiseur (3 cases)
  - Sous-marin (3 cases)
  - Destroyer (2 cases)

## Mise en Place
1. Chaque joueur dispose ses navires sur sa grille, sans que l'adversaire ne voie. Les navires peuvent être placés horizontalement ou verticalement, mais jamais en diagonale.
2. Les navires ne peuvent pas se chevaucher ou dépasser les limites de la grille.

## Déroulement du Jeu
1. Les joueurs tirent au sort pour déterminer qui commence.
2. À tour de rôle, chaque joueur annonce une case (par exemple, B5) sur la grille de son adversaire où il pense qu'un navire est situé.
3. L'adversaire répond par "touché" si la case contient une partie d'un de ses navires, ou "à l'eau" si la case est vide.
4. Si toutes les cases d'un navire sont touchées, le joueur annonce "coulé".

## Fin du Jeu
Le jeu se termine lorsqu'un joueur a coulé tous les navires de l'adversaire. Ce joueur est alors déclaré vainqueur.

## Stratégies
- Tentez de disperser vos navires pour rendre plus difficile leur localisation par l'adversaire.
- Utilisez des tirs systématiques pour découvrir et détruire les navires de votre adversaire plus efficacement.

Bon jeu et que le meilleur stratège gagne !
